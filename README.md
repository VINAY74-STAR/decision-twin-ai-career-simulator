# Decision Twin: AI-Powered Career Simulator

## 📌 Overview
This project helps users choose the right career path using AI-based decision simulation.

## 💡 Problem
Students often feel confused while choosing careers due to lack of guidance.

## 🚀 Solution
This system takes user skills as input and suggests suitable career options.

## 🛠 Tech Stack
- Python
- Basic AI Logic

## ✨ Features
- Career prediction
- Simple interface
- Fast results

## ▶️ How to Run
1. Open terminal
2. Run: python app.py
3. Enter your skills
